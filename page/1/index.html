Here we go again. Inspired by Magnus Romnes I decided to setup my own blog that will probably never be updated. The cool thing is that this blog is driven by Blogofile, a static website compiler written in Python. Why is that cool, yoy ask?

It is cool because static HTML is safe, there's nothing executable and no databases like when using Wordpress, where security holes are discovered all the time.

It is cool because it allows me to write all my blogposts in the bestest editor known to man, the allmighty Vim.

It is cool as by hosting this site on Github there's version control included, and publishing this post is a simple as a git push command.
